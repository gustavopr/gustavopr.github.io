Using SIFT with Python

Recently, I've being toying with image comparison algorithms in Python and, when I tried to use SIFT, supposedly availabe in opencv, I dicovered that it was not there!

Searching for clues, I discovered that SIFT is a patented algo, so it was removed from that library. But it was once available, and you can use an old version of the lib in order to use it.

In Linux/macOS/Windows, you can use opencv up to version 3.4.2.17 and still use SIFT:

pip install opencv-python==3.4.2.17 opencv-contrib-python==3.4.2.17
